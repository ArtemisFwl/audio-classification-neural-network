# Audio Classification using Neural Networks

## Project Overview

This project classifies audio signals into multiple categories using MFCC (Mel Frequency Cepstral Coefficients) feature extraction and a Neural Network built with TensorFlow/Keras.

## Technologies Used

- Python
- NumPy
- Pandas
- Librosa
- TensorFlow / Keras
- Scikit-Learn
- Matplotlib

## Dataset

- Dataset: UrbanSound8K
- Total Audio Samples: 8732
- Number of Classes: 10
- Feature Extraction Method: MFCC (Mel Frequency Cepstral Coefficients)
